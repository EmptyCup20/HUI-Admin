# HUI-Admin

## 启动项目
- `npm install` 安装项目依赖
- `node app.js` 启动项目
- `localhost:8080` 访问项目，默认端口为8080

## 注意事项
- 运行项目需确保HUI项目已启动，且HUI-Admin的public文件夹下的login.js和app.js里的ajax路径ip和端口需改为当前运行的HUI服务端的ip与端口
